## **micrograd(++)**
reimplementation of [Karpathy's micrograd](https://github.com/karpathy/micrograd) in C++. built purely for educational purposes.

1. engine.h : tiny autograd engine that implements backpropogation.
2. nn.h : neural networks library built on top of autograd engine with (PyTorch-like) API.
3. main.cpp : test case -- feel free to tweak.
